# Cerbege | My First JS Project 🚀

Welcome to my very first Vanilla JavaScript milestone! This is a minimalist, modern, and fully responsive landing page designed to demonstrate core front-end development skills, dynamic UI state management, and interaction handling without any external frameworks.

## ✨ Features

- **Dark / Light Mode Toggle:** Seamlessly switches the entire website's color palette using raw JavaScript and CSS custom variables.
- **Modern & Minimalist Design:** Built with modern CSS layout techniques (Flexbox & custom tracking) to achieve a high-end showcase.
- **Fully Responsive:** Beautifully optimized for mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack & Concepts Covered

- **HTML5:** Semantic structuring (`<main>`, `<header>`, `<section>`).
- **CSS3:** Custom properties/variables (`:root`), Flexbox alignment, smooth `transition` handling, and hardware-accelerated animations (`translateY`).
- **Vanilla JavaScript:** 
  - `DOMContentLoaded` listener to ensure safe DOM execution.
  - State manipulation via HTML custom attributes (`data-theme`).
  - Dynamic `textContent` updates.
